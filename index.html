What is iRduino?

iRduino is an interface, and control, application between the iRacing API and an Arduino board. iRduino can control displays, LEDs, buttons and much more. iRduino also supports custom iRacing Arduino projects by easily passing selected iRacing variables to the Arduino board.

iRduino started out as application for running the TM1638 & TM1640 display units. iRduino is highly capable in this regard and if you are an iRacer and own either of these display units then this application is for you.

System Requirements

The following applications/items are required:

  • An iRacing subscription.
  • An Arduino board (recommend boards are: the Uno, the Leonardo, and the Due).
  • Microsoft .NET 4.0 framework, or newer.
  • SlimDX
